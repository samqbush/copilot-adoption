## Local Development
- Uses [jekyll-gfm-admonitions](https://github.com/Helveg/jekyll-gfm-admonitions) for GitHub flavored alerts

```shell
# Remove unused gems
bundle clean --force

# Install gems & run local server
bundle install
bundle exec jekyll serve
```
